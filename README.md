[open app on your browser (codesandbox)](https://codesandbox.io/p/devbox/github/FaberVitale/vite-elm-template)

## Development guide

### Scripts

#### Start project

```bash
npm i
```

#### Dev server

```bash
npm run dev
```

#### Build

```bash
npm run build
```

#### run tests

```bash
npm run test
```

#### format

```bash
npm run fmt
```

#### run linters

```bash
npm run lint:elm && npm run lint:ts
```
